# 台灣大老二 AI (Taiwan Big Two AI) 隱私權政策

最後更新日期：2026年5月14日

## 1. 簡介
「台灣大老二 AI」（以下簡稱「本應用程式」）是一款為研究與娛樂目的開發的撲克牌遊戲。我們非常重視您的隱私，並致力於保護在使用本應用程式過程中所涉及的任何資訊。

## 2. 無數據收集
我們 **不會** 收集、儲存或傳送任何個人資料、遊戲數據或識別資訊至我們的伺服器。所有的遊戲進度、AI 學習筆記以及系統設定均僅儲存在您的本地設備中（使用 LocalStorage）。

## 3. 第三方 AI 介接
本應用程式允許使用者連接至大型語言模型 (LLM) API（如 LM Studio、OpenAI 或其他相容的本地/遠端伺服器）。
- **使用者責任**：當您設定 API 網址與金鑰時，本應用程式會將遊戲狀態（手牌、歷史紀錄）發送至該特定端點以進行戰略分析。
- **數據處理**：我們無法控制第三方服務提供商如何處理您的數據。請參閱您所選用的 API 提供商之隱私權政策。

## 4. 本地儲存 (Local Storage)
本應用程式會使用您設備的本地儲存空間來儲存：
- 遊戲設定（語言、角色選擇）。
- AI 設定（API 網址、模型 ID）。
- AI 學習紀錄（JSON 記憶檔案）。
您可以隨時透過清除應用程式資料或手動刪除本地檔案來清除這些資訊。

## 5. 聯絡我們
如果您對本隱私權政策有任何疑問，請透過我們的 GitHub 儲存庫與我們聯絡。
